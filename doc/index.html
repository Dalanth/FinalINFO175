Documentación Tarea_Final_INFO175¶

Esta aplicación muestra una lista de animales, cada una con sus respectivos datos e imagen.

Se requiere de:

  • Linux Mint/Ubuntu

  • Python 2.7 (viene en el sistema operativo

  • PySide:
    • Pasos para instalación:
      1. #sudo add-apt-repository ppa:pyside
      2. #sudo apt-get update
      3. #sudo apt-get install python-pyside
      4. #sudo apt-get install pyside-tolls
